Privacy Policy — 4:6 Coffee Brew Timer

Effective date: July 21, 2026

4:6 Coffee Brew Timer ("the app") is developed by yopachara. This policy explains what data the app handles and how.

Data collection

The app has no user accounts and no sign-in. It does not collect any data you type or any brew data you create — that stays on your device (see below). It does use Firebase, a Google service, for crash reporting, performance monitoring, and basic usage analytics, described below.

Data stored locally

Brew settings and history (coffee weight, ratio, taste balance, and timestamps you create while using the timer) are stored only on your device, in a local database. This data never leaves your device and is not accessible to the developer or any third party. Uninstalling the app deletes this data.

Analytics and diagnostics (Firebase)

The app uses the following Firebase services, provided by Google, to help keep the app stable and understand basic usage:

  • Firebase Crashlytics — collects crash logs and diagnostic device info when the app crashes, so problems can be found and fixed.
  • Firebase Performance Monitoring — collects diagnostic data such as app startup time and rendering performance.
  • Firebase Analytics — collects basic app interaction data (such as screens viewed) and a device/app instance identifier, not tied to your name, email, or any account.

This data is sent to and processed by Google in accordance with the Firebase Privacy and Security policy. It is transmitted encrypted (HTTPS/TLS). It is used only to operate, diagnose, and improve the app — not for advertising, and not sold or shared for any other purpose.

Permissions

  • Notifications — to show the running brew timer and let you control it from the notification shade.
  • Foreground service — to keep the countdown timer running reliably while you brew.
  • Network state and boot-completed — required by Android's WorkManager library, which the app uses for scheduling.

The app also uses network access to send the Firebase diagnostic data described above.

Third parties

The only third-party service integrated is Firebase (Google), for crash reporting, performance monitoring, and analytics as described above. The app does not integrate any advertising SDKs and does not sell or share data for marketing purposes.

Data deletion

Uninstalling the app deletes all locally stored brew data immediately. Firebase diagnostic data tied to your device's install instance can be requested for deletion by contacting the developer below.

Children's privacy

The app does not knowingly collect personal information from children. It has no accounts and no way to identify who is using it.

Changes to this policy

If this policy changes, the updated version will be posted at this same URL with a revised effective date.
